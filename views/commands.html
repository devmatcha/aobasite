Core

info

Arguments: None
Description: Gives info about Aoba such as version number, date created, API, and social media.
Usage: >info

help

Arguments: Command name (optional)
Description: Shows list of commands and allows you to view more information about each command
Usage: >help, >help blacklist

Moderation

autorole

Arguments: Role (optional), ON or OFF (optional)
Description: Sets the given role to be automatically assigned to new members. Enter ON or OFF instead of a role in order to either turn autorole on or off. This command requires the Manage roles permission.
Usage: >autorole User, >autorole ON

ban

Arguments: User mention, "Reason" (optional)
Description: Bans the given user from the server. This command requires the Ban members permission.
Usage: >ban @BadUser, >ban @BadUser "For being a troll"

blacklist

Arguments: Delete [word] (optional), Word (optional)
Description: Allows you to view the server's blacklist. Use >blacklist [word] to add a word to the blacklist and >blacklist delete [word] to remove a word from the blacklist. This command requires the Manage server permission.
Usage: >blacklist, >blacklist kill, >blackist delete kill

channel

Arguments: Name (optional), Topic (optional)
Description: Allows you to view information about the current channel. Use >channel name [name] to change the name of the channel and >channel topic [topic] to change the topic. This command requires the Manage channels permission.
Usage: >channel, >channel name talking-channel, >channel topic Talk in this channel.

emoji

Arguments: Image URL, Name
Description: Create a custom emoji. This command requires the Manage emojis permission.
Usage: >emoji http://imagehost.com/happyface.png happyface

kick

Arguments: User mention, "Reason" (optional)
Description: Kicks the given user from the server. This command requires the Kick members permission.
Usage: >kick @BadUser, >kick @BadUser "For being a troll"

prefix

Arguments: Desired prefix (optional)
Description: View the current prefix of the server. Use >prefix [desired prefix] to change the prefix of the server. This command requires Administrator privileges.
Usage: >prefix, >prefix a!

prune

Arguments: Number
Description: Delete a given amount of messages from the channel. This command requires the Manage messages permission.
Usage: >prune 20

selfrole

Arguments: Role name (optional), add [role name] (optional), delete [role name] (optional)
Description: View a list of the server's self-assignable roles.
Use >selfrole [role name] to assign yourself a role.
Use >selfrole add [role name] and >selfrole delete [role name] to add or delete roles from the list of self-assignable roles. The add and delete commands require the Manage roles permission.
Usage: >selfrole Red, >selfrole add Blue, >selfrole delete Green

server

Arguments: roles, name, icon
Description: View information about the server.
Use >server roles to view a list of roles in the server. >server roles [role name] allows you to view information about the given role.
Use >server name [name] to change the name of the server and >server icon [icon url] to change the server icon.
The name and icon commands require the Manage server permission.
Usage: >server, >server roles, >server roles Moderator, >server name The Lone Zone, >server icon http://imagehost.com/servericon.png

unban

Arguments: Username (not a mention), Reason (optional)
Description: Unban a user from the server. This command requires the Ban members permission.
Usage: >unban ForgivenMember, >unban ForgivenMember "They're valid, I guess."

Utility

flipcoin

Arguments: None
Description: Flip a coin!
Usage: >flipcoin

invite

Arguments: None
Description: Sends an invite link to invite Aoba to other servers
Usage: >invite

join

Arguments: None
Description: Causes Aoba to join the voice channel you are currently in. This command will not work if you are not in a voice channel.
Usage: >join

leave

Arguments: None
Description: Causes Aoba to leave the voice channel you are currently in. This command will not work if you are not in a voice channel.
Usage: >leave

notes

Arguments: add [note] (optional), delete [number or all] (optional)
Description: Your own personal notepad.
Use >notes add [note] to add a note to your list.
Use >notes delete [number] to delete a specific note or >notes delete all to delete all notes.
You can hold a maximum of five notes.
Usage: >notes, >notes add Clean my room, >notes delete, >notes delete 4

parrot

Arguments: Text to repeat
Description: Aoba will repeat the text that you give.
Usage: >parrot I'm a parrot!

parrotd

Arguments: Text to repeat
Description: Aoba will repeat the text that you give and delete your original message.
Usage: >parrotd I'm a parrot!

Fairies Story

All Fairies Story commands begin with >fairiesstory or >fs.

(no command)

Arguments: None
Description: Opens your profile.
Usage: >fs

start

Arguments: None
Description: Create your Fairies Story profile.
Usage: >fs start

species

Arguments: change [desired species] (optional)
Description: View your species.
Use species change [desired species] to change your species. The current available species are: Human, Fairy, Gnome, Elf, Dragonborn, Tiefling, and Genasi.
Usage: >fs species, >fs species change Fairy

fp

Arguments: give [amount] @[user] (optional)
Description: View how much FP, or money, you have.
Use fp [numeral amount] @[user] to give FP to another user.
Usage: >fs fp, >fs fp 1000 @Friend

stats

Arguments: None
Description: View your stats.
Usage: >fs stats

items

Arguments: sell (optional)
Description: View your inventory.
Use items sell [item] to sell an item and items sell [item] x[number] to sell an item in bulk.
Usage: >fs items, >fs items sell Yummy Food, >fs items sell Yummy Food x3

gamble

Arguments: amount
Description: Gamble a given amount of money away.
Usage: >fs gamble 2000

gacha

Arguments: None
Description: Use the Fairy Gacha Machine for 500FP.
Usage: >fs gacha

rep

Arguments: @User
Description: Give someone a reputation point. You earn 100FP for giving someone a reputation point. This command can only be used once a day.
Usage: >fs rep @Friend

set

Arguments: Channel name
Description: Set the channel where Fairies Story notifications are sent (for example, leveling up and picking up items).
Use set any to reset the channel. This command requires the Manage server permission.
Usage: >fs set bot-spam, >fs set any

reward

Arguments: big/small, @User
Description: Reward a user in your server for doing a good deed. This command can only be used once an hour and requires Administrator privileges.
Usage: >fs reward big @Saint, >fs reward small @GoodGuy

Music

All music commands start with >music. These commands will not work if you are not inside of a voice channel.

play

Arguments: Search term or YouTube URL
Description: Enters a voice channel and plays the requested music
Usage: >music play Thnks Fr Th Mrmrs by Fall Out Boy

queue

Arguments: None
Description: View the queue.
Usage: >music queue

volume

Arguments: Number
Description: Change the volume to a desired amount. Suggested amount is between 2 and 7.
Usage: >music volume 5

pause

Arguments: None
Description: Pause the playing song.
Usage: >music pause

resume

Arguments: None
Description: Resume the paused song.
Usage: >music resume

skip

Arguments: None
Description: Skip the current song
Usage: >music skip

stop

Arguments: None
Description: Stops the music player and makes Aoba leave.
Usage: >music stop
